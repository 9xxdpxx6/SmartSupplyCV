# GUI приложение для анализа зон магазина

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
streamlit run gui_app.py
```

Приложение откроется в браузере по адресу `http://localhost:8501`

## Возможности

- ✅ Загрузка видеофайлов через интерфейс
- ✅ Добавление зон вручную (через координаты)
- ✅ Загрузка зон из файла zones.json
- ✅ Сохранение зон в файл
- ✅ Запуск анализа видео
- ✅ Отображение результатов в виде таблицы
- ✅ Визуализация результатов с тепловой картой

## Для интерактивного выделения зон мышкой

Для полноценного drag & drop выделения зон используйте скрипт:

```bash
python setup_zones.py
```

Этот скрипт открывает окно OpenCV с полной поддержкой:
- Выделения прямоугольников мышкой (зажать ЛКМ и перетащить)
- Отображения выделенных зон в реальном времени
- Ввода названий зон
- Сохранения зон в файл

## Альтернативы

### Streamlit (текущее решение)
- ✅ Простота использования
- ✅ Красивый интерфейс из коробки
- ✅ Легкая интеграция с Python
- ❌ Drag & drop на изображении требует кастомного компонента

### Gradio
- ✅ Хорошая поддержка ML-приложений
- ✅ Встроенные компоненты для изображений
- ❌ Drag & drop на изображении также требует кастомного решения

### Кастомная GUI (PyQt/Tkinter)
- ✅ Полный контроль над интерфейсом
- ✅ Нативная поддержка drag & drop
- ❌ Больше времени на разработку
- ❌ Более сложный код

## Рекомендация

Для MVP рекомендуется комбинированный подход:
1. **Streamlit GUI** для основного функционала (загрузка видео, анализ, просмотр результатов)
2. **setup_zones.py** для точного интерактивного выделения зон мышкой

Это дает лучшее из обоих миров: удобный веб-интерфейс и точное выделение зон.
